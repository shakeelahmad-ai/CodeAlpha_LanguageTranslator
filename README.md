# 🌍 CodeAlpha Language Translator

A professional Language Translator Tool built with Python and Streamlit for the CodeAlpha AI Internship.

## 🚀 Features

- Translate text into multiple languages
- Simple and user-friendly interface
- Text-to-Speech (TTS)
- Copy translated text
- Translation history
- Fast and accurate translations

## 🛠️ Technologies Used

- Python
- Streamlit
- deep-translator
- gTTS
- Pyperclip

## 📂 Project Structure

```
CodeAlpha_LanguageTranslator/
│── app.py
│── config.py
│── translator.py
│── utils.py
│── requirements.txt
│── README.md
│── history.json
│── .gitignore
│── assets/
│── screenshots/
```

## ⚙️ Installation

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
